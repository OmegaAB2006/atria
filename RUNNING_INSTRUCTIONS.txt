TROUBLESHOOTING:
----------------
If port 5000 is busy:
    lsof -ti:5000 | xargs kill -9

If port 3000 is busy:
    lsof -ti:3000 | xargs kill -9

If backend has errors:
    cd backend
    source venv/bin/activate
    pip install -r requirements.txt
    python app.py

If frontend has errors:
    cd frontend
    npm install
    npm start
